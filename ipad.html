iPad Pro

The new iPad Pro features a 12.9-inch design with a resolution of 2732 x 2048 . It can have a full-size keyboard on it, but the width is basically similar to the height of the previous iPad Air, allowing you to use it like a MacBook Pro.

In terms of performance, the new iPad Pro uses the A9X processor, which is twice as fast as the previous A8X processor. The graphics processor is comparable to the current desktop computer, and the lifetime is about 10 hours.

In addition, Smart Keyboard has been added to the new iPad Pro, which is an external physical keyboard in addition to the protection function. This keyboard not only has a stable keyboard design, but also through magnetic links, the iPad Pro can decide whether to enable a virtual keyboard or a physical keyboard.

Even without an external keyboard, the size and weight of the iPad Pro itself is a contradiction. With a huge 12.9-inch screen, it can be difficult to type with one hand or pick up the device with one hand. Indeed, the iPad Pro has more powerful processors and runs up to 4 GB of memory to support higher-quality software, but these software may be exclusive to the iPad Pro because other iPads can't run.
